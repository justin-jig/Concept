

javascript 탄생

* 1995년 넷스케이프 커뮤니케션즈는 웹페이지의 보조적인 기능을 수행하기 위해
* 브라우저에서 동작하는 경량 프로그래밍 언어
* 브렌던 아이크가 개발한 자바스크립트 언어
* 1996년 3월 넷스케이프 네비게이터2 모카
* 1996년 9월 라이브스크립트로 이름이 변경
* 1996년 12월 자바스크립트 최종 명명
* 이후 javascript 파생 버전이 Jscript 출시

javascript 표준화
* 1996년 8월 마이크소프트 jscript 인터넷익스플로러 3.0에 탑재
JScript와 자바스크립트가 표준화도지 못하고 적당히 호환되었음
=> 넷스케이프 커뮤니케이션즈와 마이크로소프트는 자사 브라우저의 시장 점유율을
 높이기 위해 자사 브라우저에서만 동작하는 기능을 경쟁적으로 추가
=> 이로 인해 브라우저 따라 동작하는 브라우저 따라 크로스 브라우징 이슈가 발생
=> 크로스 브라우징 이슈에 따라 표준의 필요성이 요구
* 1996년 11월 넷스케이프 커뮤니케이션즈는 표준화를 위해 비영리 ECMA 인터내셔널
* 1997년 7월 ECMA-262 (ECMAScript 1)
