2. 컴퓨터 프로그램의 한 속성으로, 서로 분리되어 작성되는 성질


library 도서관
1. 소프트웨어 개발에 쓰이는 하부 프로그램의 모임이다.
2. 다양한 위치의 콘텐츠를 한 데 모아놓은 가상 폴더이다.

특정 기능에 대한 도구 또는 함수들의 집합

소프트웨 프레임워크
2. 프로그래밍에서 특정 운영 체제를 위핸 응용 프로그램 표준 구조를 구현하는 
클래스와 라이브러리 모임

웹 프레임워크
 1 동적인 웹 페이지나, 웹 애플리케이션, 웹 서비스 개발 보조용으로 만들어지는 애플리케이션 프레임워크의 일종이다.
 2. 웹 페이지를 개발하는 과정에서 겪는 어려움을 줄이는 것이 주 목적으로 통상 데이터베이스 연동, 템플릿 형태의 표준, 세션 관리, 코드 재사용 등의 기능을 포함하고 있다.
